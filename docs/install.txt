###### DEPLOYING KALI VM ENVIRONMENT ######
1.) SSH to the Kali VM:
ssh kali@<VM_IP_ADDRESS>

2.) Install Docker on VM (one-time setup):
sudo apt update
sudo apt install docker.io -y
sudo systemctl enable --now docker
sudo usermod -aG docker $USER
newgrp docker

3.) Copy project files into Kali (Git):
git clone https://github.com/jackaizik/VAPT-Scanner.git VAPT-Scanner
cd VAPT-Scanner

4.) Build and Run Docker Container:
docker build -t vapt-scanner .
docker run -d -p 5000:5000 -p 8080:8080 vapt-scanner

###### BUILD AND RUN ######
1.) Activate python environment:
pip install -r requirements.txt

2.) Run Locally (on Windows):
python app.py
